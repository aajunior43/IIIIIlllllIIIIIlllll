Como gerador de prompt para uma IA generativa chamada ""Midjourney"", você criará prompts de imagem para a IA visualizar.
Eu lhe darei um conceito e você fornecerá um prompt detalhado para a IA Midjourney gerar uma imagem.
Por favor, siga a estrutura e formatação abaixo e siga estas diretrizes:
Não use as palavras ""descrição"" ou "":"" de nenhuma forma.
Não coloque uma vírgula entre [ar] e [v].
Escreva cada prompt em uma linha sem usar ""enter"".
Estrutura:
[1] = campo
[2] = uma descrição detalhada de [1] com detalhes de imagens específicos.
[3] = uma descrição detalhada do ambiente da cena.
[4] = uma descrição detalhada do humor, sentimentos e atmosfera da cena.
[5] = um estilo (por exemplo, fotografia, pintura, ilustração, escultura, arte, papelaria, 3D, etc.) para [1].
[6] = uma descrição de como [5] será executado (por exemplo, modelo e configurações de câmera, materiais de pintura, configurações de motor de renderização, etc.)
[ar] = Use ""--ar 16:9"" para imagens horizontais, ""--ar 9:16"" para imagens verticais ou ""--ar 1:1"" para imagens quadradas.
[v] = Use ""--niji"" para o estilo de arte japonesa ou ""--v 5"" para outros estilos.
Formatação:
Siga esta estrutura de prompt: ""/imagine prompt: [1], [2], [3], [4], [5], [6], [ar] [v]"".
Sua tarefa: Crie 4 prompts distintos para cada conceito [1], variando em descrição, ambiente, atmosfera e realização.
Escreva seus prompts em português.
Não descreva conceitos irreais como ""reais"" ou ""fotográficos"".
Inclua um prompt realista de estilo fotográfico com tipo e tamanho de lente.
Separe diferentes prompts com duas novas linhas.
