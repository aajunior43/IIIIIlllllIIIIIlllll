Gostaria que você me ajudasse a criar prompts para o Midjourney. Primeiro, deixe-me explicar o que é o Midjourney e como vamos gerar prompts para ele. Também passaremos por 20 exemplos para garantir que você entenda.

Midjourney é uma ferramenta de IA que cria imagens a partir da entrada do usuário, semelhante ao DALL-E.

A parte principal do prompt são palavras ou frases que descrevem a imagem que você deseja. Mais adjetivos e substantivos descritivos específicos criam imagens únicas. Por outro lado, substantivos ou adjetivos básicos criam imagens simples. Tenha em mente que o Midjourney não entende gramática. Portanto, prompts muito longos podem não funcionar bem. Ao criar prompts, remova quaisquer palavras desnecessárias. Menos palavras dão a cada palavra mais importância, garantindo que a imagem esteja alinhada com o seu tema.

Por exemplo, "ilustre para mim um belo pôr do sol sobre um oceano sereno, faça as cores quentes e suaves, e renderize-o em um estilo impressionista". Este prompt tem palavras que o Midjourney pode não entender ou trabalhar. Frases como "Ilustre para mim" são desnecessárias. Verbos como "fazer" e "renderizar" também são redundantes. O Midjourney geralmente aceita palavras descritivas como substantivos e adjetivos. O prompt poderia ser mais simples: "pôr do sol quente e suave sobre oceano sereno, pintura a óleo impressionista".

Sinônimos mais específicos geralmente funcionam melhor do que os gerais. Por exemplo, use palavras precisas como "pequeno", "compacto", "diminuto" e "minúsculo" em vez de "pequeno". Ao criar seu prompt, concentre-se nos detalhes específicos que você deseja:

- **Tema:** Pessoas, animais, lugares, personagens, objetos, eventos, etc.
- **Uso:** Logos, design de sites, design de interiores, protótipos, design de produtos, etc.
- **Mídia:** Fotos, desenhos, ilustrações, pinturas, desenhos animados, aquarelas, colagens, esculturas, grafites, mosaicos, tapeçarias, cerâmica, etc.
- **Ambiente:** Interno, externo, cidade, floresta, ilha, deserto, debaixo d'água, caverna, cidade futurista, espaço, lua, estação espacial, etc.
- **Iluminação:** Suave, crepúsculo, ambiente, luz solar direta, nublado, luz da lua, neon, luz de vela, luz de fogo, etc.
- **Cor:** Vibrante, suave, brilhante, monocromático, colorido, preto e branco, pastel, etc.
- **Humor:** Energético, sedado, calmo, ruidoso, inquieto, melancólico, sonhador, misterioso, etc.
- **Composição:** Retrato, close-up, headshot, vista aérea, simétrico, linhas principais, minimalista, silhueta, panorama, etc.
- **Artistas:** Van Gogh, Picasso, D

ali, Paul Cézanne, Leonardo da Vinci, Botticelli, Rembrandt, Hayao Miyazaki, etc.
- **Estilos de arte:** Pintura de paisagem oriental, Ukiyo-e, arte conceitual, Bauhaus, Impressionismo, Rococó, Surrealismo, longa exposição, etc.

Você também pode usar uma vírgula, sinal de mais ou "e" para separar diferentes assuntos. Por exemplo, para representar uma luz e uma casa, você deve separá-las. Caso contrário, se você digitar "luz casa", o Midjourney mostrará um farol para você.

Os prompts do Midjourney também podem incluir parâmetros para alterar a forma como uma imagem é gerada. Os parâmetros comuns incluem:

1. **--ar**

   Este parâmetro define a proporção da tela. O padrão é 1:1; `--ar 16:9` para telas de computador e `--ar 9:16` para telas de celular.

2. **--v**

   Este parâmetro especifica a versão do Midjourney. `--v 5.2` representa a versão 5.2.

3. **--s**

   Este parâmetro controla o nível artístico da imagem, com valores de 0 a 1000. Um valor mais alto indica uma imagem mais artística, mas a imagem também se desviará mais do efeito real descrito pelo prompt. Valores mais baixos indicam que o efeito da imagem está mais próximo da descrição do prompt, mas menos artístico. A configuração sugerida é `--s 750`.

4. **--style**

   Este parâmetro `--style raw` é usado para imagens fotográficas.

5. **--q**

   Este parâmetro define o detalhe da imagem. Mais detalhes são criados com valores maiores. O valor padrão é `--q 1`. A configuração recomendada é `--q 2`.
